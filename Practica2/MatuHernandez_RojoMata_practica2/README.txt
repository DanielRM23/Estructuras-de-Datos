La práctica consiste en la implementación de una clase llamada "Conjunto" utilizando arreglos de tipo entero (positivo).
Dicha clase simula el uso de estos objetos matemáticos implementando los siguientes métodos: 
1.- Método constructor.
2.- Método esVacio.
3.- Método pertenencia.
4.- Método subConjunto.
5.- Método diferencia.
6.- Método union.
7.- método intersección.
8.- Método toString.

## Compilación y Ejecución

1. Asegurárse de tener Java JDK instalado en el sistema.
2. Descargar o clonar este archivo en la computadora.
3. Abrir la terminal y navegar hasta la carpeta del proyecto.

### Compilación

Ejecutar el siguiente comando para compilar el proyecto:

javac *.java
