# Ejercicio 9, Implementación de la Matriz de Adyacencias

# Rojo Mata Daniel

Implementación en Java de los siguientes métodos para la EDD Gráfica
1.- esVecino
2.- regresaMatrizAdyacencias

La clase Gráfica contiene la implementación de ambos métodos. 
Mientras tanto, la clase PruebaMatriz incluye el método main que se 
utiliza para ejecutar un ejemplo de cómo se utilizan los métodos implementados.

## Compilación y Ejecución

1. Asegurárse de tener Java JDK instalado en el sistema.
2. Descargar o clonar este archivo en la computadora.
3. Abrir la terminal y navegar hasta la carpeta del proyecto.

### Compilación

Ejecutar el siguiente comando para compilar el proyecto:

javac *.java
