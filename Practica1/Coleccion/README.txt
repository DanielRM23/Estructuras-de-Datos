La práctica consiste en una implementación de la interfaz genérica "Coleccion" en Java, que proporciona métodos para agregar, eliminar, acceder y buscar elementos en una colección genérica.
Además de que se incluyen dos métodos adicionales, a saber; Tamanio y esVacia

## Compilación y Ejecución

1. Asegúrate de tener Java JDK instalado en tu sistema.
2. Descarga o clona este archivo en tu computadora.
3. Abre una terminal y navega hasta la carpeta del proyecto.

### Compilación

Ejecuta el siguiente comando para compilar el proyecto:

javac *.java
